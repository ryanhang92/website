CS61C - Convolutional Neural Network Optimization

 

CS61C - Convolutional Neural Network Optimization

In the final course of Berkeley's introductory CS series I was apply able with achieve a 5x performance speedup of a convolutional neural network trained to identify cats in images without the use of multiple machines. This project allowed me to tie together my understanding of memory, cache usage, paralleism in computing, how C code is compiled into assembly, and other topics in lower-level computing.

Basic Ballot - Political Sentiment Analyzer

 

Basic Ballot - Political Sentiment Analyzer

I built the first protoype of the Political Sentiment Analyzer during my work with Basic Ballot and this was my first attempt at applying rudimentary machine learning. This project applied Latent Dirichlet Allocation (LDA) and text sentiment analyzers to dicover how candidates felt about political topics by parsing through a collections of news articles scraped from the web. This experience continues to drive my interst in applying Computer Science to political questions.
